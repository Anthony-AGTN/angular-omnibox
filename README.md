# Angular omnibox

Omnibox searchbar for angular

## Prerequisites

* AngularJS 1.5+
* [Font Awesome](https://fontawesome.com/)

## Installing
Get module:
```
npm install --save angular-omnibox
```

## Example
### build
```
npm install
npm run example
```
### run
open [example/index.html](example/index.html) in browser

### code
see [examples](example/example.js)

## Running the tests

Unit tests:
```
npm run units
```

Lesshint tests:
```
npm run lesshint
```

CS tests:
```
npm run cs
```
